# Paper

The paper component `sp-paper` is a surface component, that can be used for showing block content lifted from the main page background.

<Canvas>
  <Story name="Default">${html` <sp-paper>Paper content</sp-paper> `}</Story>
</Canvas>

## Level

Optionally set a stacking/elevation level to adjust the drop shadow applied. Defaults to `1`.

<Canvas>
  <Story name="Level: 1">${html` <sp-paper level="1">Level 1</sp-paper> `}</Story>
  <Story name="Level: 2">${html` <sp-paper level="2">Level 2</sp-paper> `}</Story>
</Canvas>

## Border

Optionally set the `has-border` attribute to apply a subtle border around the element.

<Canvas>
  <Story name="Border">${html` <sp-paper has-border>Border</sp-paper> `}</Story>
</Canvas>
