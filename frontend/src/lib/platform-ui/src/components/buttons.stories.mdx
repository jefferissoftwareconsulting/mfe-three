# Buttons

Use buttons to trigger actions.

## Style

There are four styles of button: `primary`, `secondary`, `transparent` and `transparent-primary` that are set with the `button-type` property.

<Canvas>
  <Story name="Style: Primary">${html`<sp-button button-type="primary">Primary</sp-button>`}</Story>
  <Story name="Style: Secondary">${html`<sp-button button-type="secondary">Secondary</sp-button>`}</Story>
  <Story name="Style: Transparent">${html`<sp-button button-type="transparent">Transparent</sp-button>`}</Story>
  <Story name="Style: Transparent Primary">
    ${html`<sp-button button-type="transparent-primary">Transparent Primary</sp-button>`}
  </Story>
</Canvas>

## Disabled

Buttons can be disabled using the `disabled` property.

<Canvas>
  <Story name="Disabled: Primary">${html`<sp-button button-type="primary" disabled>Primary</sp-button>`}</Story>
  <Story name="Disabled: Secondary">${html`<sp-button button-type="secondary" disabled>Secondary</sp-button>`}</Story>
  <Story name="Disabled: Icon">${html`<sp-button leading-icon="emoji" disabled></sp-icon>`}</Story>
  <Story name="Disabled: Transparent">
    ${html`<sp-button button-type="transparent" disabled>Transparent</sp-button>`}
  </Story>
  <Story name="Disabled: Transparent Primary">
    ${html`<sp-button button-type="transparent-primary" disabled>Transparent Primary</sp-button>`}
  </Story>
</Canvas>

## Size

There are two sizes of button - normal and compact (using the `compact` property).

<Canvas>
  <Story name="Size: Normal">{html`<sp-button>Normal</sp-button>`}</Story>
  <Story name="Size: Large">{html`<sp-button compact>Compact</sp-button>`}</Story>
</Canvas>

## Icons

Buttons can have an icon either as a prefix or a suffix. Use `leading-icon`/`trailing-icon`. Icon size can be customised via the `icon-size` attribute.

<Canvas>
  <Story name="Leading-icon">
    ${html`<sp-button leading-icon="jobs" button-type="transparent" compact>Leading Icon</sp-button>`}
  </Story>
  <Story name="Trailing-icon">
    ${html`<sp-button trailing-icon="jobs" button-type="transparent" compact>Trailing Icon</sp-button>`}
  </Story>
  <Story name="Both Icons">
    $
    {html`<sp-button leading-icon="jobs" trailing-icon="jobs" button-type="transparent" compact>Both Icons</sp-button>`}
  </Story>
  <Story name="Icon Only">${html`<sp-button leading-icon="emoji" compact></sp-button>`}</Story>
  <Story name="Icon Only Small">${html`<sp-button leading-icon="emoji" icon-size="6" compact></sp-button>`}</Story>
</Canvas>

## React

To consume the `Button`/`Button Group` components in a React app, import it from the `@skedulo/platform-ui-react` package and use it as you would a normal react component.
Kebab cased properties will be mapped to camelcase.

<Source language="tsx" code={dedent`
import { Button, ButtonGroup } from '@skedulo/platform-ui-react'

<ButtonGroup compact>
  <Button button-type="primary" onClick={event => {}} loading compact leadingIcon="icon" trailingIcon="icon">
    I am a button!
  </Button>
</ButtonGroup>
`}/>
