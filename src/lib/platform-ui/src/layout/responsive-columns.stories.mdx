# Responsive Columns

Component that lays out children as columns until the `x-small` breakpoint (`60rem`) when the columns are stacked. Padding can be configured with the `--sp-padding` CSS variable.

<Canvas>
  <Story name="Responsive Columns">
    $
    {html`
      <sp-responsive-columns style="background-color: var(--sp-color-neutral-300)">
        <div style="padding: var(--sp-spacing-4); background-color: var(--sp-color-purple-300)">Column 1</div>
        <div style="padding: var(--sp-spacing-4); background-color: var(--sp-color-green-300)">Column 2</div>
      </sp-responsive-columns>
    `}
  </Story>
</Canvas>
