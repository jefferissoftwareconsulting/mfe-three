# Badge

The Badge component `<sp-badge>`.

Badges can contain numeric values which can indicate a count, such as the number of unread notifications.
Badges can also contain letters/words, such as `alpha`, `beta`, or `new`.
When containing numbers and the limit has been hit, the limit with a `+` sign will be shown.
When containing a string and the limit has been hit, the string upto the limit with a `..` will be shown.

## Theme

There are four themes: `default`, `subtle`, `primary` and `important`. Changed via the `theme` property.

## Content

Takes `content` property which can be a number or string

## Limit

Takes `limit` property which defaults to `99` for numbers and `5` for strings

## React

To consume the `Badge` in a React app, simply import it from the `@skedulo/platform-ui-react` package and use it as you would a normal React component.

<Source
  code={dedent`
    import { Badge } from '@skedulo/platform-ui-react'
    <Badge content="11" limit="9" theme="primary"  />
  `}
/>
