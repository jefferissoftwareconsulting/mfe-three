## Dropdown Button

`sp-combo-box` allows you to create a dropdown and takes a `sp-button` as it's first slot.

  <Story height="290px" name="Default">

        <div slot="options">
          <sp-combo-box-option>Option 1</sp-combo-box-option>

## Dropdown options

The dropdown can have a custom amount of options set by using `sp-combo-box-option` and can be binded with the `@click` property.
