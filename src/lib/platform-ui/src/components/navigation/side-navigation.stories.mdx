# React

To consume `sp-side-navigation-item` in a React app import it from the `@skedulo/platform-ui-react` package.

<Source
  code={dedent`
  import { SideNavigationItem } from '@skedulo/platform-ui-react'
  <SideNavigationItem url="https://www.test.com">Label</RecordRow>
`}
/>
