# Icons

Icons can be rendered with the `sp-icon` component that takes an `icon` property (see available icons below) and a `size` property (default is 18px)

<Canvas>
  <Story name="Icon">${html`<sp-icon icon="close" size="40"></sp-icon>`}</Story>
</Canvas>

## Color

Icons use the current color.

<Canvas>
  <Story name="Color">
    $
    {html`
      <div style="color: var(--sp-color-primary-600);">
        <sp-icon icon="infoFill"></sp-icon>
        Information!
      </div>
    `}
  </Story>
</Canvas>
