# Typography

## Headings

The `sp-heading` should be used for headings with a `size` property set to `3xl`, `2xl`, `xl`, `lg`, `base`, or `sm`.
The `level` property can optionally be set to 1-6 to choose which of the h1-h6 tags are used.
Note: no padding or margin is included on this component so that it can be used in as many places as possible.

## Body sizes

Body text size is set using CSS variables. Actual sizes are provided here to help translate from designs.

| Size        | CSS Variables                               | Actual size |
| :---------- | :------------------------------------------ | :---------- |
| Default     | `--sp-font-size` / `--sp-line-height`       | 14px / 20px |
| Small       | `--sp-font-size-sm` / `--sp-line-height-sm` | 13px / 16px |
| Extra Small | `--sp-font-size-xs` / `--sp-line-height-xs` | 12px / 15px |

## Font weights

Font weights are set using CSS variables. Actual weights are provided here to help translate from designs.

| Weight    | CSS Variable                | Actual weight |
| :-------- | :-------------------------- | :------------ |
| Bold      | `--sp-font-weight-bold`     | 700           |
| Semi-bold | `--sp-font-weight-semibold` | 600           |
| Medium    | `--sp-font-weight-medium`   | 500           |
| Normal    | `--sp-font-weight-normal`   | 400           |

<Canvas>
  <Story name="Font weights">
    $
    {html`
      <div style="font-weight: var(--sp-font-weight-bold); margin-bottom: var(--sp-spacing-4);">bold</div>
      <div style="font-weight: var(--sp-font-weight-semibold); margin-bottom: var(--sp-spacing-4);">semibold</div>
      <div style="font-weight: var(--sp-font-weight-medium); margin-bottom: var(--sp-spacing-4);">medium</div>
      <div style="font-weight: var(--sp-font-weight-normal);">normal</div>
    `}
  </Story>
</Canvas>
