# Colors

Design system colors are made available via css custom properties. They are all namespaced under `--sp-color-...`. The suffixed number i.e. `-900` represents the shade, higher being darker. To use reference `var(--sp-color...)` in the css of your element.

If a given color doesn't match your brand palette it can be overriden by redefining the css custom property on the `sp-global-styles` element.

```css
sp-global-styles {
  --sp-color-primary-600: #ffcc00;
}
```
